# Dependency Injection

Basic implementation for dependency injection pattern

## Run the example

- Download example files:

```bash
$ git clone https://github.com/MohamedElmdary/dependencyInjection
```

- then install dependencies:

```bash
$ yarn install
```

`OR`

```bash
$ npm install
```

- Run the example

```bash
    $ yarn start
```

`OR`

```bash
    $ npm run start
```
